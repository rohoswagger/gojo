# Security Policy

## Reporting a Vulnerability

If you discover a security issue in Gojo, please report it through the GitHub Security Advisory flow:

https://github.com/rohoswagger/gojo/security/advisories/new

Please include:
- a clear description of the issue
- reproduction steps
- affected macOS version(s)
- any suggested mitigation if you have one

Third-party dependency vulnerabilities should also be reported upstream to the relevant package maintainers.

If the issue relates to licensing, attribution, or source-distribution
compliance for Gojo or one of its bundled third-party components, open a
regular repository issue instead of using the security advisory flow.
