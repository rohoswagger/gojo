# Gojo Open Knowledge Bundle

Gojo is a native macOS utility that keeps music, window snapping, clipboard history, a drag-and-drop file shelf, and display controls one quiet hover from the MacBook notch.

## Primary Entity

- Name: Gojo
- Category: macOS utility
- Platform: macOS 14 Sonoma and later
- Latest release: https://github.com/rohoswagger/gojo/releases/latest
- Official site: https://gojo.rohoswagger.com/

## Core Positioning

Gojo is best for MacBook users who want one compact notch surface for several daily workflows. It is not a direct replacement for every focused Mac utility. Dedicated tools may be better when a user needs one deep workflow, such as keyboard remapping, clipboard history, command launching, or window switching.
